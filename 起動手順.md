# YouTube文字起こしアプリ 起動手順

このプロジェクトの起動コマンドは `start_app.sh` に統一します。

## 1. 事前準備

```bash
cd /home/ohide/usr8_work/work_23_chatgpt/16_PoCs/0106_cc-sdd
export OPEN_API_KEY="sk-proj-あなたの有効なキー"
```

- `OPEN_API_KEY` は `start_app.sh` 内で `OPENAI_API_KEY` として扱われます。
- 文字起こし実行まで確認する場合は、有効な OpenAI キーが必須です。

## 2. 起動コマンド

### バックエンドのみ

```bash
./start_app.sh --no-build
```

### バックエンド + フロントエンド(Docker)

```bash
./start_app.sh --with-frontend --no-build
```

### バックエンド + フロントエンド(ローカルNode)

```bash
./start_app.sh --frontend-local --no-build
```

## 3. 動作確認

### APIヘルスチェック

```bash
curl http://localhost:8000/health
```

期待値:

```json
{"status":"healthy","database":"healthy","redis":"healthy"}
```

### UI確認

ブラウザで以下を開く:
- http://localhost:3000
- http://localhost:8000/docs

## 4. 停止

```bash
docker compose down
```

## 5. 補助スクリプト

`run02_app.sh` は `start_app.sh` を呼び分けるショートカットです。

```bash
./run02_app.sh --help
./run02_app.sh backend
./run02_app.sh docker-frontend
./run02_app.sh local-frontend
```
